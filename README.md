# GreenClamAV
GreenClamAV is a portable, green antivirus software for Windows, built on the world-renowned open-source antivirus engine ClamAV. It comes in a single compact folder and automatically updates its virus database upon launch. It can serve as an emergency antivirus tool and is completely free!
GreenClamAV 是一款基于全球知名开源杀毒引擎 ClamAV 打造的、适用于 Windows 系统的绿色便携版杀毒软件。它仅需一个小巧的文件夹即可运行，启动时自动升级病毒库，可作为应急防病毒工具使用，且完全免费！
